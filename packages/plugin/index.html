Orbinex dev

Chat bubble is in the bottom-right corner. Click it to open.

Make sure the engine is running at http://localhost:3001 and the MCP server at http://localhost:3002 before chatting.

Try asking

  • "What is the weather in Mumbai?" — live weather via open-meteo
  • "Tell me about Tokyo" — city tour tool
  • "What is the capital of France?" — country capital tool
  • "What is the exchange rate of USD to INR?" — live currency tool
  • "Show me experience data" — hits your local API at localhost:5017
  • "What is Orbinex?" — searches your docs via RAG
